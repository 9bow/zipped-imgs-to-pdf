Features

  • Convert ZIP files containing images to PDF
  • Convert multiple image files to a single PDF
  • Configurable Natural Sort - sorts numbers in filenames naturally (e.g., page_1, page_2, page_10)
  • Priority character settings - place images starting with specific characters first (default: !)
  • Supports various image formats (JPG, JPEG, PNG, GIF, WebP)
  • 100% client-side processing - files are never uploaded to a server
